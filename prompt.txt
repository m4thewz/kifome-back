Quero que você gere um código em Node.js para preencher meu banco de dados com vários dados iniciais. O projeto é um site de receitas e utiliza services para criar registros no banco, ou seja, para inserir qualquer dado basta chamar o service correspondente passando um JSON com os atributos necessários. Não posso usar seeders nem rodar comandos no terminal, então o código deve ser executado apenas uma vez e depois removido. As tabelas do sistema incluem usuário, receita, categoria, comentário, like e ingrediente, e é importante considerar que para criar uma receita é necessário já existir um usuário autor, portanto o código precisa respeitar a ordem correta de criação dos dados e armazenar os IDs retornados quando necessário. Com os services e os atributos exigidos por cada um, gere um arquivo completo em Node.js contendo uma função principal que chama os services e cria vários registros reais de exemplo para cada tabela, exibindo logs durante o processo, evitando duplicações caso seja executado acidentalmente e tratando erros de forma simples. No final, entregue somente o código completo dentro de um bloco js e use apenas as informações que eu enviar sobre os services, sem pedir dados adicionais.
